# 02. 機能仕様書（生徒側）

## 1. ダッシュボード
- **ステータス表示**: レベル、累計EXP、連続学習日数（ストリーク）。
- **直近サマリー**: 過去1週間の正答率、頻出エラー傾向のフィードバック。
- **クイックスタート**: AIが現在のタグ熟練度から選出した「次の最適な1問」への直行ボタン。

## 2. コーディング・ワークスペース
- **エディタ**: Monaco Editor（VSCode準拠）、シンタックスハイライト、コード補完。
- **AIサイドバー**: ヒント要求、質疑応答（直接の答えは教えないプロンプト制御）。
- **コンソール**: 実行結果、翻訳済みエラーメッセージ（行数ハイライト付き）。

## 3. リザルト＆レビュー
- **EXPアニメーション**: 獲得項目の内訳表示。
- **PRレビュー**: 正解後、AIシニアエンジニアによるコード品質向上のためのコメント。
- **コミュニティ解答**: 厳選された3件（最速、最短、最新）の匿名コード閲覧。
