# 04. 技術インフラ仕様書

## 1. インフラ構成 (GCP)
- **APIサーバー**: Cloud Run (FastAPI)
- **データベース**: Cloud SQL (PostgreSQL), Cloud Memorystore (Redis)
- **ストレージ**: Cloud Storage (静的ファイル/ビルド物)

## 2. セキュア・サンドボックス
- **ランタイム**: Docker + gVisor (ホストカーネルからの隔離)。
- **ウォームスタンバイ**: 言語ごとに起動済みコンテナをプール（Redisで管理）し、実行遅延を1秒未満に抑制。
- **リソース制限**: タイムアウト(2秒)、メモリ(128MB)、ネットワーク(完全遮断)。

## 3. AI連携ロジック
- **マルチLLM**:
    - Gemini (Vertex AI): 問題生成、解説、一般的ヒント（低コスト・高速）。
    - GPT-4o (OpenAI): 高度なコードレビュー、複雑な論理判定（高精度）。
- **非同期処理**: Celery + Redis によるバッチ処理およびポーリング方式による結果取得。
